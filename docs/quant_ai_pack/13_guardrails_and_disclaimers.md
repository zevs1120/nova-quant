# Guardrails and Disclaimers

## Product guardrails
- Default mode favors clarity over complexity
- Risk reduction beats forced action
- Stand-aside is a valid high-quality output

## Model guardrails
- No release without audit logs
- No release without cost-aware validation
- No release without expected trade density declaration

## Communication guardrails
Avoid language implying:
- guaranteed profit
- certainty
- personalized fiduciary relationship unless legally structured
- hidden leverage or risk

## Internal reminder
A great consumer quant product should make users calmer, not more impulsive.
