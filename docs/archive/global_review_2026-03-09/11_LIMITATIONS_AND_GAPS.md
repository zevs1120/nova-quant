# Nova Quant Global Review — Limitations and Gaps

As of 2026-03-08.

## A) Research Weaknesses

1. Runtime discovery registries are narrower than available seed libraries.
2. Some strategy-family breadth exists mostly in metadata/templates, not equally in empirically replayed behavior.
3. Research-material doctrine is strong, but policy enforcement in runtime modules is still partial.
4. Discovery throughput and promotions are generated from simulation-driven candidate metrics, limiting real confidence.

## B) Validation Weaknesses

1. Walk-forward and robustness frameworks are architecturally solid but still fed by proxy/synthetic paths in key places.
2. Shadow opportunity forward outcomes are synthetic rather than bar-joined realized paths.
3. Cost/slippage modeling is still simplified and not venue/liquidity-regime calibrated.
4. Fill realism remains weak in many runs (for example low filled/roundtrip conversion in weekly report snapshots).

## C) Portfolio Weaknesses

1. Portfolio simulator uses deterministic proxy expected-return/volatility dynamics.
2. Correlation matrix is heuristic/synthetic rather than historical covariance-based.
3. Regime-aware allocation logic is present but not yet stress-validated with full event-level replay.
4. Portfolio-level claims can look stronger than underlying empirical evidence quality.

## D) Product Weaknesses

1. UX maturity has improved faster than research realism maturity.
2. User-facing conclusions can appear authoritative while upstream evidence is partly synthetic.
3. AI explanations are structured and useful, but not yet tied to high-fidelity realized outcome tracking.
4. Some server/product paths still rely on mock datasets (`public/mock/*`).

## E) Engineering / Due-Diligence Weaknesses

1. Test suite is meaningful but mostly contract/smoke-level; deep robustness regression tests are limited.
2. Human governance workflow (reviewers, approvals, signed memos) is not fully operationalized.
3. Adapter layer for broker/exchange connectivity is mostly mock in current implementation.
4. Front-end bundle size warning indicates maintainability/performance optimization work remains.
5. Repository is credible for early-stage DD, but not yet production-audit-grade for execution realism claims.

## Skeptic Summary

A serious reviewer will likely accept Nova Quant as a promising early platform, but remain skeptical until:

- synthetic paths are replaced by realized replay paths,
- governance becomes operationally strict,
- and portfolio/validation outputs are empirically hardened.
