# Archive Policy

Last updated: 2026-03-23

This folder stores historical audits/reviews that are preserved for traceability only.

Rules:

1. Archived files must start with a clear historical warning.
2. Archived files are not authoritative for current runtime/diligence status.
3. Canonical current-state docs remain in the top-level `docs/` directory (see [`../REPOSITORY_OVERVIEW.md`](../REPOSITORY_OVERVIEW.md) and [`../REPO_RUNBOOK.md`](../REPO_RUNBOOK.md)).
